# 📔 AJ Organizer Pro - The Open Book Vault

**AJ Organizer Pro** is a dual-purpose digital diary and security vault designed for mobile portrait view. It looks like a simple daily routine organizer, but it hides a powerful **Media Vault** and an **Intruder Capture System**.

## 🚀 Key Features
* **Portrait Optimized:** Perfect "Open Book" experience for mobile users.
* **Intruder Trap:** Silently captures a photo after failed login attempts.
* **Persistent Notepad:** Your daily routines are saved locally.
* **Hidden Vault:** Store images/videos securely using browser-based storage.
